# Развёртывание бота на Railway (для клиента)

Инструкция, как развернуть собственную копию бота на своём аккаунте Railway.
Отдельная база данных (Postgres) не нужна — используется SQLite на диске.

---

## Что понадобится заранее

1. **Telegram-бот** — создать через @BotFather, получить токен вида `123456:ABC...`
2. **Сообщество ВК** (если нужен VK-бот) — токен сообщества с правами:
   сообщения + управление + работа со стеной
3. Аккаунт на **github.com** и **railway.com**

---

## Шаг 1. Аккаунт Railway

1. Зайти на railway.com, войти через GitHub
2. Привязать банковскую карту (Railway требует её даже для триала;
   на триале $5 бесплатных кредитов на 30 дней)

## Шаг 2. Создать проект из репозитория

1. New Project → Deploy from GitHub repo
2. Выбрать репозиторий с ботом
   (если репозиторий приватный — сначала получить к нему доступ у владельца)
3. Railway начнёт первую сборку по Dockerfile

## Шаг 3. Добавить постоянный диск (Volume)

Без этого база данных сотрётся при каждом обновлении.

1. Открыть сервис → вкладка Settings → раздел Volumes
2. Add Volume, mount path указать: **/data**

## Шаг 4. Переменные окружения

Открыть сервис → вкладка Variables → добавить (Raw Editor удобнее):

```
EDITION=lite
DATABASE_URL=sqlite+aiosqlite:////data/badminton.db
TG_TOKEN=сюда_токен_от_BotFather
TG_MODE=polling
VK_TOKEN=сюда_токен_сообщества_ВК
RUN_VK_POLLING=true
JWT_SECRET=любая_случайная_строка
ADMIN_API_TOKEN=любая_случайная_строка
TIMEZONE=Europe/Moscow
LOG_DIR=/data/logs
PORT=8080
```

Примечания:
- Если VK-бот не нужен — оставить `VK_TOKEN` пустым и `RUN_VK_POLLING=false`
- `EDITION=lite` — базовый набор (запись, очередь, напоминания, посещаемость).
  Для экспорта, статистики, оплат и веб-админки поставить `EDITION=pro`

## Шаг 5. Запуск и проверка

1. После сохранения переменных Railway передеплоит сервис
2. Дождаться статуса Active в логах
3. Открыть Telegram-бота, отправить `/start` — бот должен ответить
4. Для ВК: написать сообществу — бот должен ответить

---

## Обновление бота

Когда владелец обновит код в репозитории, Railway автоматически пересоберёт
и перезапустит сервис. Данные в /data сохраняются.

## Про стоимость

- Триал: $5 кредитов на 30 дней — хватает для теста
- Дальше для постоянной работы нужен план Hobby ($5/месяц)
- Точные тарифы: railway.com/pricing
