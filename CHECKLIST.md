# Чеклист боевой проверки (15 минут)

Прощёлкать после деплоя, чтобы убедиться, что всё живое.

## Telegram (личка бота)
- [ ] /start — приветствие, меню внизу (у админа расширенное)
- [ ] 🏸 Тренировки — карточки, запись/отмена кнопкой
- [ ] 📅 Мои записи — все ваши записи со статусом
- [ ] 🏆 Рейтинг — топ посещаемости + строка «Вы»
- [ ] 📊 Статистика (админ) — рейтинг, должники, сводка по месяцам, архив, график
- [ ] ➕ Создать тренировку — мастер кнопками до конца
- [ ] ✏️ у карточки — изменить время → записанным пришло уведомление
- [ ] 🗑 Отменить тренировку — подтверждение, уведомления, кнопки исчезли
- [ ] 📆 Расписание — добавить шаблон, изменить поле, удалить
- [ ] ⏰ Напоминание — сменить время, вернуть обратно
- [ ] 📢 Рассылка — текст → подтверждение → пришло и в ТГ, и в ВК

## ВКонтакте (личка сообщества)
- [ ] «начать» — приветствие (если задано) + список тренировок
- [ ] Запись/отмена, 🔄 Обновить список
- [ ] 📅 Мои записи, 🏆 Рейтинг
- [ ] Создание тренировки кнопками, ✏️ Изменить, 🔁 Повторить
- [ ] 📊 Статистика (админ) + кнопка «📄 CSV» — файл пришёл
- [ ] 📆 Расписание, ⏰ Напоминание, ✏️ Имена

## Кросс-платформенность
- [ ] Запись из ВК видна в карточке ТГ (и наоборот)
- [ ] Создание из ВК → карточка в ТГ-группе + анонс на стене ВК

## Веб (браузер)
- [ ] /club/1 — страница клуба, тренировки, имена записанных
- [ ] Запись по имени+телефону → ссылка отмены работает
- [ ] «Мои записи» по телефону внизу страницы
- [ ] /club/1/qr — открывается QR-картинка

## Админка (браузер, роль owner)
- [ ] /admin — дашборд, ссылки на публичную страницу и QR
- [ ] 💾 Скачать бэкап — скачивается .db
- [ ] 🛠 Конфигуратор — собирается архив клиента с .env и seed.db
- [ ] ⚙️ Настройки клуба сохраняются

## Фон (проверяется по факту)
- [ ] За N минут до тренировки записанным пришло «⏰ Скоро тренировка»
- [ ] Тренировка из расписания создалась сама за N дней
- [ ] В /data/backups появился суточный бэкап (SQLite)

## Мультиклиент (при подключении клиента на общий сервер)
- [ ] PATCH /tokens вернул "reloaded": true — боты клиента поднялись без рестарта
- [ ] /start у клиентского бота отвечает и показывает ЕГО клуб
- [ ] Запись в клиентском боте не появляется в тренировках первого клуба
- [ ] Напоминание участнику клиента приходит от ЕГО бота
- [ ] PATCH /billing с прошедшей датой: бот клиента отвечает «приостановлено», после очистки даты — снова работает
